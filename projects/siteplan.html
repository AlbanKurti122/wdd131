Interactive Albania Guide - Site Plan

Site Name

Interactive Albania Guide

This site represents a comprehensive guide for travelers exploring Albania. It combines information about destinations, cultural activities, and travel planning tools in one interactive platform.

Optional domain: interactivealbania.com

Site Purpose

The site provides an interactive and dynamic guide to Albania. Visitors can explore popular destinations, filter activities by interest (nature, culture, beaches), and plan trips using a built-in travel form. The goal is to inform and engage users while demonstrating practical JavaScript, HTML, and CSS features.

Scenarios

  • Scenario 1: A traveler wants to find the best cultural cities to visit in Albania. Which destinations should they explore?
  • Scenario 2: A user wants to create a personal travel plan and see a summary of chosen cities and activities.

Color Schema

  • #1B4F72 – Primary color for headings, navigation, and borders
  • #FDFEFE – Background color for pages
  • #F39C12 – Accent color for buttons, links, highlights

Typography

  • Poppins, sans-serif – Used for headings (h1, h2, h3)
  • Roboto, sans-serif – Used for body text and paragraphs
  • Monospace – Optional for code examples or highlights

JavaScript Functionality Plan

  • Filter destinations based on user-selected interest (DOM manipulation)
  • Dynamic display of destination cards using arrays & template literals
  • Form submission handling with validation
  • Store selected destinations and activities in localStorage
  • Display travel summary dynamically on Plan Your Trip page
  • Use multiple functions, conditional statements, and array methods (filter, map)
